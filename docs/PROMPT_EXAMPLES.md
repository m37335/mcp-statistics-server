# よくある指標を使ったプロンプト例

## 📊 GDP関連

### 1. 日米のGDPを折れ線グラフで比較
```
日米のGDPを折れ線グラフで比較してください。期間は2020年から2023年までです。
```

**期待される結果:**
- アメリカと日本のGDP推移を時系列で表示
- 2020-2023年の4年間のデータ
- 折れ線グラフで2つの系列を比較

### 2. G7諸国のGDPを棒グラフで比較
```
G7諸国（アメリカ、日本、ドイツ、イギリス、フランス、イタリア、カナダ）の2023年のGDPを棒グラフで比較してください。
```

**期待される結果:**
- 7カ国の2023年GDPを棒グラフで表示
- 国別の比較が一目で分かる
- 降順で並べ替え

### 3. G5諸国のGDP構成比を円グラフで表示
```
G5諸国（アメリカ、日本、中国、ドイツ、イギリス）の2023年のGDP構成比を円グラフで表示してください。
```

**期待される結果:**
- 5カ国のGDP構成比を円グラフで表示
- 各国のシェアが視覚的に分かる
- パーセンテージ表示

## 💡 プロンプト作成のコツ

### 良いプロンプトの特徴

1. **明確な目的**: 「何を比較したいか」を明確に
   - ✅ 「日米のGDPを比較」
   - ❌ 「データを見たい」

2. **具体的な期間**: 期間を指定
   - ✅ 「2020年から2023年まで」
   - ❌ 「最近のデータ」

3. **チャートタイプの指定**: 目的に応じたタイプを選択
   - **折れ線グラフ**: 時系列データの推移
   - **棒グラフ**: カテゴリ間の比較
   - **円グラフ**: 構成比の表示

4. **国名の明記**: 国コードではなく国名で指定可能
   - ✅ 「アメリカ、日本、中国」
   - ✅ 「G7諸国」
   - ✅ 「主要先進国」

### プロンプト例のテンプレート

```
[対象国]の[指標名]を[チャートタイプ]で[比較/表示]してください。[期間指定]
```

**例:**
- 「日米のGDPを折れ線グラフで比較してください。期間は2020年から2023年までです。」
- 「G7諸国の2023年のGDPを棒グラフで比較してください。」
- 「主要国のGDP構成比を円グラフで表示してください。」

## 🔍 よく使われる指標コード

| 指標名 | コード | 説明 |
|--------|--------|------|
| GDP（現行価格） | `NY.GDP.MKTP.CD` | 国内総生産（USドル） |
| GDP成長率 | `NY.GDP.MKTP.KD.ZG` | 実質GDP成長率（%） |
| 一人当たりGDP | `NY.GDP.PCAP.CD` | 一人当たりGDP（USドル） |
| 人口 | `SP.POP.TOTL` | 総人口 |
| インフレ率 | `FP.CPI.TOTL.ZG` | 消費者物価指数の変化率（%） |
| 失業率 | `SL.UEM.TOTL.ZS` | 失業率（%） |
| 貿易額（GDP比） | `NE.TRD.GNFS.ZS` | 貿易額のGDPに対する割合（%） |

## 📝 使用時の注意点

1. **データの可用性**: 指定した期間や国にデータが存在しない場合があります
2. **最新データ**: データは各APIの更新頻度に依存します
3. **国コード**: システムは国名から自動的に国コードを変換しますが、不明な場合は国コードを直接指定することも可能です
